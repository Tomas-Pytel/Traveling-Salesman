# UNIX_SEMESTRALKA_PYTEL

Táto semestrálna práca sa zaoberá vytvorením aplikácie s ktorou je možné komunikovať cez CLI.
Táto aplikácia slúži na riešenie úlohy Obchodného cestujúceho pomocou Pažravej metódy, Metódy Zdvojenia kostry a Metódy kostry a párenia v uliciach mesta Čadca.
1. Po spustení aplikácie je používateľ vyzvaný, aby zadal koľko náhodných adries z mesta sa má vygenerovať
2. Aplikácia vypíše na kozolu, ktorý algoritmus našiel akú cestu spolu s jej dĺžkou.
3. Aplikácia tiež uloží výsledky algoritmov do samostatných súborov, v ktorých je možné si pozrieť trasu, ktorú daný algoritmus naplánoval na mape
4. Po vypísaní výsledkov sa aplikácia spýta používateľa, či chce pokračovať, ak áno, môže znova zadať počet adries, ak nie, potom sa aplikácia ukončí
